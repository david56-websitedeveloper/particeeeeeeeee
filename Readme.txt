१. बिल डिजाइन (Bill Design)
मुल्य (Amount):
= Quantity * Rate
व्याख्या: मात्रा (Quantity) र दर (Rate) गुणन गरेर मुल्य (Amount) निकालिन्छ।
उदाहरण: Monitor को लागि = 20 * 6660 = 133,200

छुट (Discount):
= Amount * Discount %
व्याख्या: मुल्यलाई छुट प्रतिशतले गुणन गरेर छुट रकम निकालिन्छ।
उदाहरण: यदि छुट 10% भए = 133,200 * 10% = 13,320

कुल मुल्य (Total Amount):
= Amount - Discount
व्याख्या: मुल्यबाट छुट घटाएर कुल मुल्य निकालिन्छ।
उदाहरण: 133,200 - 13,320 = 119,880

VAT (मूल्य अभिबृद्धि कर):
= Total Amount * 13/100
व्याख्या: कुल मुल्यको 13% VAT गणना गरिन्छ।
उदाहरण: 119,880 * 13% = 15,584.40

नेट मुल्य (Net Amount):
= Total Amount + VAT
व्याख्या: कुल मुल्यमा VAT जोडेर नेट मुल्य निकालिन्छ।
उदाहरण: 119,880 + 15,584.40 = 135,464.40

२. तलब शीट (Salary Sheet)
सेवा कर (Service Tax):
= Salary * 1%
व्याख्या: तलबको 1% सेवा कर लगाइन्छ।
उदाहरण: Rajan को लागि = 26,000 * 1% = 260

आयकर (Income Tax):
=IF(Salary*12<=200000, 0, IF(AND(Salary*12>200000, Salary*12<=300000), Salary*15%, Salary*20%))
व्याख्या:

यदि वार्षिक तलब 2 लाखभन्दा कम भए आयकर 0%

यदि वार्षिक तलब 2-3 लाख भए आयकर 15%

यदि वार्षिक तलब 3 लाखभन्दा बढी भए आयकर 20%
उदाहरण: Rajan को वार्षिक तलब = 26,000 * 12 = 312,000 (3 लाखभन्दा बढी)
आयकर = 26,000 * 20% = 5,200

कुल कर (Total Tax):
= Service + Income
व्याख्या: सेवा कर र आयकर जोडेर कुल कर निकालिन्छ।
उदाहरण: 260 (सेवा कर) + 5,200 (आयकर) = 5,460

नेट तलब (Net Salary):
= Salary - Total Tax
व्याख्या: तलबबाट कुल कर घटाएर नेट तलब निकालिन्छ।
उदाहरण: 26,000 - 5,460 = 20,540

३. मिति गणना (Date Calculation)
कुल वर्ष (Total Year):
=INT((End Date - Start Date)/365)
व्याख्या: अन्त्य मितिबाट सुरु मिति घटाएर दिनको संख्या निकालिन्छ र 365 ले भाग गरि वर्ष निकालिन्छ।
उदाहरण: Pagan Acharya को लागि = (12-May-2005 - 15-Feb-1998) / 365 ≈ 7 वर्ष

कुल महिना (Total Month):
=INT((End Date - Start Date)/30)
व्याख्या: दिनको संख्यालाई 30 ले भाग गरेर महिना निकालिन्छ।
उदाहरण: उही मितिहरूको लागि ≈ 87 महिना

हप्ता (Week):
=INT((End Date - Start Date)/7)
व्याख्या: दिनको संख्यालाई 7 ले भाग गरेर हप्ता निकालिन्छ।
उदाहरण: ≈ 379 हप्ता

दिन (Day):
=INT(End Date - Start Date)
व्याख्या: सिधा दिनको संख्या निकालिन्छ।
उदाहरण: ≈ 2,653 दिन

४. मजदुरी गणना (Wage Calculation)
मजदुरी (Wages):
=IF(Hours Worked>8, (8*200)+(Hours Worked-8)*400, Hours Worked*200)
व्याख्या:

यदि 8 घण्टाभन्दा कम काम गरे: घण्टा * 200

यदि 8 घण्टाभन्दा बढी काम गरे: पहिलो 8 घण्टा 200 को दरले, बाँकी 400 को दरले
उदाहरण:

Ram (12 घण्टा): (8*200) + (4*400) = 1,600 + 1,600 = 3,200 Rs

Shyam (6 घण्टा): 6 * 200 = 1,200 Rs

५. बिजुली बिल (Electricity Bill)
खपत युनिट (Con Unit):
=C Unit - P Unit
व्याख्या: हालको युनिटबाट अघिल्लो युनिट घटाएर खपत युनिट निकालिन्छ।
उदाहरण: Ram Acharya को लागि = 430 - 320 = 110 युनिट

बिल रकम (Bill Amount):
=IF(Con Unit<=20, 80, IF(Con Unit<=250, 80+(Con Unit-20)*7.7, 80+(230*7.7)+(Con Unit-250)*9.9))
व्याख्या:

20 युनिटसम्म: फिक्स्ड 80 Rs

20-250 युनिट: 80 + (अतिरिक्त युनिट * 7.7)

250 भन्दा बढी: 80 + (230*7.7) + (250 भन्दा बढी * 9.9)
उदाहरण:

Ram Acharya (110 युनिट): 80 + (90*7.7) = 80 + 693 = 773 Rs

Nikita Brandari (500 युनिट): 80 + (230*7.7) + (250*9.9) = 80 + 1,771 + 2,475 = 4,326 Rs

६. नेपाल टेलिकम बिल (Nepal Telecom Bill)
कुल कल (Total Call):
= Current Call - Previous Call
व्याख्या: हालको कलबाट अघिल्लो कल घटाएर कुल कल निकालिन्छ।
उदाहरण: Rajan को लागि = 1503 - 1250 = 253 कल

अतिरिक्त कल (Extra Call):
= Total Call - 100
व्याख्या: 100 कलसम्म फिक्स्ड छ, त्यसभन्दा बढी कललाई अतिरिक्त कल मानिन्छ।
उदाहरण: Rajan को लागि = 253 - 100 = 153 कल

रकम (Amount):
= 100 + (Extra Call * 2)
व्याख्या: 100 कलको फिक्स्ड 100 रुपैयाँ + प्रत्येक अतिरिक्त कलको 2 रुपैयाँ।
उदाहरण: Rajan को लागि = 100 + (153*2) = 100 + 306 = 406 रुपैयाँ

VAT (मूल्य अभिबृद्धि कर):
= Amount * 13%
व्याख्या: रकमको 13% VAT गणना गरिन्छ।
उदाहरण: 406 * 13% = 52.78 रुपैयाँ

नेट रकम (Net Amount):
= Amount + VAT
व्याख्या: रकममा VAT जोडेर नेट रकम निकालिन्छ।
उदाहरण: 406 + 52.78 = 458.78 रुपैयाँ

७. बोनस, भत्ता र कर गणना (Bonus, Allowances & Tax Calculation)
बोनस (Bonus):
=IF(Salary<=10000, Salary*7%, 0)
व्याख्या: यदि तलब 10,000 भन्दा कम वा बराबर छ भने तलबको 7% बोनस।

उदाहरण: Kalyan Siha (गढघट): (8000+560)*12% = 1,027.20 रुपैयाँ

दैनिक भत्ता (DA):
=IF(Address="Dharan", (Salary+Bonus)*9%, 0)
व्याख्या: यदि ठेगाना धरान हो भने तलब+बोनसको 9% DA।
उदाहरण: Madan Siha (धरान): (12500+0)*9% = 1,125 रुपैयाँ

फील्ड भत्ता (FA):
=IF(Address<>"Dharan", Salary*10%, 0)
व्याख्या: यदि ठेगाना धरान होइन भने तलबको 10% FA।
उदाहरण: Ram Bashel (काठमाडौं): 12000 * 10% = 1,200 रुपैयाँ

आवास भत्ता (HA):
=IF(Address<>"Gadjhat", (Salary+Bonus+TA)*15%, 0)
व्याख्या: यदि ठेगाना गढघट होइन भने तलब+बोनस+TA को 15% HA।
उदाहरण: Ram Bashel: (12000+0+0)*15% = 1,800 रुपैयाँ

कर (Tax):
=IF(Address<>"Bulwal", (Salary+Bonus+TA+DA+FA+HA)*10%, 0)
व्याख्या: यदि ठेगाना बुलवाल होइन भने कुल आयको 10% कर।
उदाहरण: Ram Bashel: (12000+0+0+0+1200+1800)*10% = 1,500 रुपैयाँ

कुल रकम (Total):
= Salary + Bonus + TA + DA + FA + HA - Tax
व्याख्या: सबै भत्ता जोडेर कर घटाउने।
उदाहरण: Ram Bashel: 12000 + 0 + 0 + 0 + 1200 + 1800 - 1500 = 13,500 रुपैयाँ

= Total Income - Total Expenditure
